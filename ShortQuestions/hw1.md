## 7.List ##

 ### unordered list ###

Syntax: -/*/+ space

- item1
- item2
- item2



### ordered list

1. item1
2. item2
3. item3



### nested list

* Item 1
  * nested item 2
    * nested item 3



### task list ###

Syntax: - space [X]/[ ]

- [x] completed task 
- [ ] task
